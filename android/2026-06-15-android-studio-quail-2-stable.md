# Android Studio Quail 2 稳定版发布：并行代理工作流与内存泄漏检测

> 原文：[Android Studio Quail 2 is Stable: Multi-task with the Android Studio AI agent](http://android-developers.googleblog.com/2026/06/android-studio-quail-2-stable-features.html) | 日期：2026-07-22

Android Studio Quail 2 正式发布，带来三项重大更新。一是全面重构的 Agent Mode，支持并行运行多个代理任务并使用不同模型，大幅提升复杂任务的处理效率。二是原生集成 LeakCanary 内存泄漏检测工具到 Profiler 中，将堆分析从测试手机转移到开发电脑上执行，泄漏追踪速度提升至原来的 5 倍且无卡顿。三是 App Quality Insights（AQI）与 Agent Mode 完全集成，点击崩溃即可获得 AI 生成的摘要和修复方案，可通过"Fix with AI"按钮自动应用代码更改。此外还修复了大量 Bug 并整合了 IntelliJ 平台的最新改进。
