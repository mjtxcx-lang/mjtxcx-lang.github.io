# SwiftUI 代理技能开源发布：专治 AI 生成代码的常见"坏习惯"

> 原文：[SwiftUI Agent Skill - Write better code with Claude, Codex, and other AI tools](https://www.hackingwithswift.com/articles/282/swiftui-agent-skill-claude-codex-ai) | 日期：2026-07-22

Paul Hudson 发布了开源 SwiftUI 代理技能，用于帮助 Claude Code、Codex 和 Gemini 等 AI 编码工具识别和修复 SwiftUI 代码中的常见问题，涵盖现代 API 使用、性能优化、无障碍访问（如 VoiceOver 兼容性）、避免使用已弃用 API、防止不必要的视图重绘等。该技能基于 Hudson 此前整理的 AI 生成 Swift 代码常见问题文章，并大幅扩展了检查范围，还包括并发编程建议、设计指导和项目规范等。Hudson 指出，LLM 在编写 SwiftUI 代码时经常采用次优模式，该技能可有效提升 AI 生成代码的质量。
