# Xcode中使用AI Agent技能：让Claude和Codex写出更好的Swift代码

> 原文：[Agent skills in Xcode: How to install and use them today](https://www.hackingwithswift.com/articles/283/how-to-install-and-use-ai-agent-skills-in-xcode) | 日期：2026-07-24

Paul Hudson详细介绍了如何在Xcode中安装和使用AI Agent技能来提升AI编程效率。Agent技能是专门解决特定编码任务的强大工具，他已发布了覆盖SwiftUI、Swift Testing、Swift并发编程和SwiftData等多个领域的技能。安装流程因工具而异：对于Claude Code和Codex命令行工具，只需运行`npx skills add`命令即可；在Xcode中使用则需要在Settings的Intelligence标签页中配置内置Agent或添加自定义Agent。安装后，技能在Claude Code（命令行和桌面版）、Codex（命令行和桌面版）以及Codex for Xcode中均可使用，但Claude for Xcode需要额外配置步骤。重启应用后即可生效，AI辅助编码的质量将显著提升。
