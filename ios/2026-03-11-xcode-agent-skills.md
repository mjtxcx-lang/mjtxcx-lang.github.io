# Xcode AI 代理技能实战：让 Claude 和 Codex 写出更好的 Swift 代码

> 原文：[Agent skills in Xcode: How to install and use them today](https://www.hackingwithswift.com/articles/283/how-to-install-and-use-ai-agent-skills-in-xcode) | 日期：2026-07-22

Paul Hudson 介绍了如何在 Xcode 中安装和使用 AI 代理技能（Agent Skills）。首先需要在 Xcode 设置中进入 Intelligence 选项卡，配置内置的 Claude 或 Codex 代理，或添加自定义代理。安装技能只需在终端运行一条命令，例如 `npx skills add https://github.com/twostraws/swiftui-agent-skill --skill swiftui-pro`，安装后即可在 Claude Code、Codex 命令行和桌面版、Codex for Xcode 中同时使用。Hudson 已在 GitHub 上发布了 Swift Agent Skills 仓库，涵盖 SwiftUI、Swift Concurrency、SwiftData 和 Swift Testing 等多个方向的专用技能。
